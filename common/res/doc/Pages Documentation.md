# **Pages Documentation**

This document provides an overview of the various pages in the **Wondamart Data Solutions** application, organized by functionality.

---

## **General Pages**

* **Home Page**: Landing page introducing Wondamart’s services, emphasizing affordable mobile data bundles and a call-to-action button (“Buy Data Now”).
- **Pricing Page**: Displays pricing plans and subscription options of bundles.
- **Blog Page** *(future)*: Displays a list of blog posts and articles.
- **Blog Post Page** *(future)*: A detailed view of a single blog post.
* **About Page** *(optional extension)*: Background information about Wondamart’s mission and purpose.
* **Contact Page**: Provides contact channels including phone, email, WhatsApp, and Telegram.
* **Terms & Privacy Policy**: Outlines service terms, usage policies, and privacy commitments.
* **FAQ Page** *(future)*: Central hub for frequently asked questions and troubleshooting support.
* **404 Page**: Custom error page for invalid or broken routes.

---

## **Authentication Pages**

* **LoginPage Page**:

    * Fields: Email, Password
    * “Forgot Password” option
    * Proceed button
* **Register Page**:

    * Fields: First Name, Last Name, Gender, Age, Telephone, Email, Status, City, Region, Referral Code, Password, Confirm Password
    * Submit button to create account
* **Password Reset Page** *(future)*: For users who forget their credentials.
* **Email Verification Page** *(future)*: For verifying new accounts via email.
- **OTP Page** *(future)*: A page for users to enter a one-time password for additional security.
- **Two-Factor Authentication Page** *(future)*: A page for setting up and managing two-factor authentication.

---

## **User Account Pages**

* **Dashboard Page**:

    * Displays welcome message, wallet balance, quick actions (Buy Data, History, Deposit, Earn Rewards).
    * Account overview includes orders summary, and recent activity logs.
* **Profile Page** *(future)*: Manage personal details and account updates.
* **Settings Page** *(future)*: Manage security and preferences.
* **Notifications Page** *(future)*: Displays system alerts and updates.

---

## **Purchase & Transactions Pages**

* **Buy Data Page**:

    * Order form with fields for Network (MTN, Telecel, AT), Bundle (2-day, 30-day, non-expiry), Amount, Quantity (auto-calculated), Phone Number, Confirm Number.
    * Submit order functionality.
* **History Page**:

    * Table listing past purchases with fields: Date/Time, Order ID, Network, Bundle, Phone Number, Quantity, Amount, Status (Processing, Failed, Delivered).
* **Deposit Page**:

    * Wallet top-up via secure payment gateways.
    * Transaction history table with fields: Date/Time, Transaction ID, Type (Top-up, Order), Network, Phone Number, Amount, Status.
* **Order Confirmation Page** *(future)*: Post-purchase summary page.
* **Invoice Page** *(future)*: Provides downloadable invoices for purchases.

---

## **Rewards & Engagement Pages**

* **Earn Rewards Page**:

    * Explains referral program benefits (earn free data, unlimited potential, referral tracking).
    * Includes instructions:

        1. Share referral link
        2. Invitee signs up
        3. Invitee buys data
        4. Referrer earns redeemable points for free data
    * Displays referral statistics (total referrals, total rewards earned).
    * Referral link with copy button.
